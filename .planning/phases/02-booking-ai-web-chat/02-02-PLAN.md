GET /api/availability?shopId=X&date=YYYY-MM-DD&staffId=X&duration=30
- Returns: TimeSlot[] with { hour, minute, label, available, startTime }

POST /api/appointments
- Body: { shopId, serviceId, staffId?, date, startTime, customerName, customerPhone, customerEmail?, notes? }
- Returns: 201 with appointment object (includes service, staff, customer, shop)
- Returns: 409 if slot taken

PATCH /api/appointments/[id]
- Body: { status: "CANCELLED" | "CONFIRMED" | ... }
- Auth: service token or user JWT

GET /api/appointments/lookup?shopId=X&phone=Y
- Returns: { customer: { id, name, phone, email }, appointments: [...] }

GET /api/internal/shop-context?shopId=X
- Returns: ShopContext object with all shop data for system prompt
